# Frontend `src` structure

This file lists the folders created as a starting point for the Ride-Sharing frontend.

- `components/` - reusable UI components
  - `common/` - shared small components (buttons, inputs)
- `pages/` - top-level pages
  - `Auth/` - login/register pages
  - `Driver/` - driver-specific pages
  - `Rider/` - rider-specific pages
- `layouts/` - layout components (MainLayout, AuthLayout)
- `hooks/` - custom React hooks
- `utils/` - utility functions
- `services/` - API clients and services
- `contexts/` - React context providers
- `styles/` - global styles and design tokens
- `assets/images/` - image assets
- `assets/icons/` - icon assets
- `routes/` - route definitions
- `constants/` - app constants
- `config/` - environment/configuration helpers
- `tests/` - frontend tests

You can expand or reorganize these folders as the project grows.
